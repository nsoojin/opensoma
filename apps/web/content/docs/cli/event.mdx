SWMaestro에서 주최하는 행사 목록을 조회하고 신청할 수 있습니다. 행사 카테고리, 접수 기간, 행사 기간, 상태 정보가 포함됩니다.

## List Events

행사 목록을 조회합니다. 카테고리, 제목, 접수 기간, 행사 기간, 상태가 포함됩니다.

```bash
opensoma event list
```

## Get Event Details

특정 행사의 상세 정보를 조회합니다. `<id>`는 목록 조회 결과의 `id` 값입니다.

```bash
opensoma event get <id>
```

## Apply for Event

행사에 신청합니다. 접수 기간 내에만 신청할 수 있으며, 이미 신청한 경우 오류를 반환합니다.

```bash
opensoma event apply <id>
```
