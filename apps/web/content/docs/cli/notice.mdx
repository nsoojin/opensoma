SWMaestro 플랫폼의 공지사항을 조회합니다. 읽기 전용이며 별도의 필터 옵션은 없습니다.

## List Notices

공지사항 목록을 조회합니다. 제목, 작성자, 작성일이 포함됩니다.

```bash
opensoma notice list
```

## Get Notice

특정 공지사항의 전체 내용을 조회합니다. `<id>`는 목록 조회 결과의 `id` 값입니다. 본문은 HTML 형식으로 반환됩니다.

```bash
opensoma notice get <id>
```
