대시보드는 현재 로그인한 사용자의 기본 정보와 최근 멘토링 세션, 회의실 예약 현황을 한 번에 조회합니다. 내 상태를 빠르게 파악하거나 AI 에이전트가 컨텍스트를 수집할 때 첫 번째로 호출하기 좋은 명령어입니다.

## Show Dashboard

```bash
opensoma dashboard show
```

출력에는 이름, 역할(연수생/멘토), 소속 팀 정보, 예정된 멘토링 세션 목록, 회의실 예약 목록이 포함됩니다. `--pretty`를 붙이면 읽기 쉬운 형태로 출력됩니다.

```bash
opensoma dashboard show --pretty
```
