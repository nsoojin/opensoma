AI 에이전트가 CLI를 효과적으로 사용할 수 있도록 [Agent Skills](https://agentskills.io/)을 포함하고 있습니다. 스킬을 설치하면 Claude Code, OpenCode 같은 AI 에이전트가 opensoma CLI의 사용법을 자동으로 이해하고 올바른 명령어를 생성할 수 있습니다.

## SkillPad

Agent Skills용 GUI 앱입니다. 자세한 내용은 [skillpad.dev](https://skillpad.dev/)를 참고하세요.

[![Available on SkillPad](https://badge.skillpad.dev/opensoma/dark.svg)](https://skillpad.dev/install/opensoma/opensoma/opensoma)

## Skills CLI

터미널에서 직접 스킬을 추가합니다.

```bash
npx skills add opensoma/opensoma
```

## Claude Code Plugin

Claude Code에서 플러그인으로 설치합니다.

```bash
claude plugin marketplace add opensoma/opensoma
claude plugin install opensoma
```

또는 Claude Code 내에서:

```
/plugin marketplace add opensoma/opensoma
/plugin install opensoma
```

## OpenCode Plugin

`opencode.jsonc`에 추가합니다. 다음 번 세션부터 opensoma 스킬이 자동으로 로드됩니다.

```json
{
  "plugins": ["opensoma"]
}
```
