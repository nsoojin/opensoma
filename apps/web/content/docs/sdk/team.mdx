현재 로그인한 사용자가 속한 팀 정보를 조회합니다. 팀에 속해 있지 않은 경우 `teams`는 빈 배열을 반환합니다.

## Show

```typescript
const team = await client.team.show()
```

### Response

`currentTeams`는 현재 참여 중인 팀 수, `maxTeams`는 최대 참여 가능 팀 수입니다.

```typescript
interface TeamInfo {
  teams: Array<{
    name: string
    memberCount: number
    joinStatus: string
  }>
  currentTeams: number
  maxTeams: number
}
```

### Example

```typescript
const team = await client.team.show()

if (team.teams.length === 0) {
  console.log('소속 팀 없음')
} else {
  for (const t of team.teams) {
    console.log(`${t.name} (${t.memberCount}명) - ${t.joinStatus}`)
  }
}
```
